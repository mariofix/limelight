Want to help? Here's how...

You can always check the source code for this project, The following is currently needed to have the "first proper version":

  • Opinions: Your opinion is very important to me, if you have suggestions, ideas, complaints, they are all very welcomed.
  • Information Gathering&Filtering: There are about 8000 projects in pypi, just add the slug and the system will do the rest, but there is so much the system can guess, so you can also modify the project's metadata so the crawler can work next time.
  • Front-End: As you can see, Visual Design is not my strong suit, ANY help in this topic is very much appreciated, just send a PR.
  • Infra&SEO: The app is currently in the free tier of pythonanywhere, I intend to pay as the traffic demands it but another domain name or subdomain will be nice, and in kind of the same topic, suggestions on how to make the domain migration without suffering in the search engines.
  • Pass the word: Just pass the word, this is the best thing you can do to help.
